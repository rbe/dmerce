=======================================================
                        Config
=======================================================

Verwendung:

		c = Guardian.Config.RFC822([<dir>])
		c.Get('<section>', '<parameter>')

	<dir> = optionaler Parameter; enthaelt ein
	        Verzeichnis, wenn dmerce.cfg nicht in .

	<section> = Sektion, aus der ein Parameter
	            <parameter> ausgelesen und zurueck-
	            gegeben werden soll
