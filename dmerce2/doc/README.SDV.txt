Zu dmerce-SDV gehoeren folgende Module:

sdv.py : Das zentrale Modul mit der SDV-Klasse
lib.py : Library mit Klassen-unabhängigen Funktionen
error.py : Modul mit der Error-Klasse
data.py : Modul zur Erzeugung von Daten-Objekten
colors.py : Modul zur Erzeugung von Farb- und Farbenlisten-Objekten
font.py : Modul zur Erzeugung von Font-Objekten
title.py : Modul zur Erzeugung eines Titelfeldes
legend.py : Modul zur Erzeugung eines Legendenfeldes
graphic.py : Modul zur Erzeugung des Graphikfeldes

Die Dokumentation besteht aus:

techdoc.tex : Technische Dokumentation
userdoc.tex : Benutzer-Schnellreferenz (mit Beispiel)
read.me : dieser Überblick
